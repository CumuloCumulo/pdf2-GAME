# 感染周期模型详解

## Will Wright 的流行病学洞察

### 核心概念

游戏传播可以类比为疾病传播模型：
- **感染** = 购买游戏
- **感染周期** = 玩家兴奋到向所有人推荐游戏的时期
- **传染源** = 正在积极推荐的玩家

### 数学关系

**关键发现**：如果感染周期翻倍，感染人口增加10倍。

```
感染人口 = f(感染周期长度)
当感染周期 × 2，感染人口 × 10
```

### 实际应用

**普通游戏**：
- 感染周期：2-4周
- 玩家购买后兴奋期短，推荐行为有限

**社区驱动游戏**：
- 感染周期：数月甚至数年
- 社区成员持续推荐，因为游戏成为生活一部分

### 延长感染周期的策略

1. **持续内容更新**
   - 定期发布新内容
   - 季节性活动
   - 事件驱动更新

2. **社交互动设计**
   - 多人协作任务
   - 竞技排行榜
   - 社交分享奖励

3. **玩家生成内容**
   - UGC工具
   - 创作分享平台
   - 社区活动

4. **身份和成就系统**
   - 稀有成就
   - 独特外观
   - 社区地位

### 现代应用

**病毒式传播（Virality）**已成为社交和移动游戏成功的关键指标：
- K因子（K-factor）：每个用户带来的新用户数
- 病毒系数 > 1 表示指数增长
- 社区是提高K因子的最有效方式
