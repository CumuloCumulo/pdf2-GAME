# 游戏影响视角分析

## 适用场景
- 分析关于游戏影响的辩论
- 评估游戏对社会变化的正负面影响
- 撰写或评估关于游戏行为的报告

## 执行步骤

### 1. 分析游戏的固有属性
认识到游戏天生通过“趣味性”吸引人们。将“所有游戏行为都是有害的”这一观点标记为具有“极端哲学倾向”。

### 2. 分类长期影响观点
将观点归入以下三个类别之一：

- **极简/中立观点**：认为游戏没有持久影响，仅是“短期消遣”和“刺激来源”。
- **负面/有害观点**：认为游戏是危险的，会引发暴力倾向、导致成瘾、毁掉玩家生活。
- **正面/教育观点**：认为游戏具有“独特教育能力”，是“21世纪教育的基石”。

### 3. 承认背景复杂性
认识到“游戏如何改变我们”这一问题难以简单回答，游戏带来的社会变化既包含“好的变化”也包含“坏的变化”。

### 4. 使用精确术语描述负面行为
当描述过度或有害的游戏模式时：
- **避免**：使用“成瘾”一词，除非有明确的医学诊断。
- **推荐**：使用“问题性使用”。
- **理由**：“成瘾”是具有严格诊断标准的医学术语，广泛使用可能不准确或带有污名化色彩。“问题性使用”准确描述了行为本身（造成问题的使用），而未做出可能不存在的特定医学诊断。

## 输出格式
该观点被归类为[极简/中立、负面/有害或正面/教育]，承认游戏引起的社会变化固有的复杂性。对于负面行为描述，使用“问题性使用”术语以确保描述准确性。
