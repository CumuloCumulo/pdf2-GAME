# 概率分布详解

## 单个骰子 vs 多个骰子

### 1d20（均匀分布）
- 每个结果概率相等：5%
- 适合极端属性生成
- 结果不可预测

### 3d6（正态分布）
- 结果呈钟形曲线
- 10-11最常见（各12.5%）
- 3和18最罕见（各0.46%）
- 适合生成"平均"属性
- 结果更可预测

## 组合概率示例

### 硬币翻转序列
所有可能的3次翻转：
- HHH, HHT, HTH, HTT
- THH, THT, TTH, TTT
共8种结果

至少2次正面：4/8 = 50%

### 卡牌抽取
52张牌中：
- 人头牌（J、Q、K）：12张
- A：4张
- 钻石：13张

抽到人头牌或A：(12+4)/52 = 16/52 = 30.8%
抽到人头牌或钻石：(12+13-3)/52 = 22/52 = 42.3%
注意：减去3张人头钻石牌避免重复计算

## "至少一个"问题

### 经典问题：24次投掷中至少一个双6

方法：计算"没有双6"的概率，然后用1减去
- 单次投掷不是双6的概率：35/36
- 24次都不是双6：(35/36)^24 ≈ 0.509
- 至少一个双6：1 - 0.509 ≈ 0.491 = 49.1%

这个结果违反直觉——大多数人认为应该超过50%
