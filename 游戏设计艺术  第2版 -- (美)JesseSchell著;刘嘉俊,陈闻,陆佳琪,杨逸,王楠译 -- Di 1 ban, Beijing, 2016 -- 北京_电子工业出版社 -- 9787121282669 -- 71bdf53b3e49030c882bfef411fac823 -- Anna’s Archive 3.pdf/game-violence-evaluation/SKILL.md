# 游戏暴力评估

## 使用时机
在以下情况下使用此技能：
- 设计或评估包含暴力元素的游戏
- 进行游戏内容分级审查
- 分析游戏对玩家心理的潜在影响
- 为家长或决策者提供暴力内容评估

## 评估流程

### 第一步：识别暴力类型
根据视觉表现和现实关联度，将暴力内容分类为：

**抽象暴力**
- 高度抽象化，无血腥画面
- 与现实世界残忍行为无关
- 例子：国际象棋、围棋
- 接受度：普遍接受

**卡通暴力**
- 风格化、远离现实
- 例子：海盗题材的卡通战斗
- 接受度：通常可接受

**图形/写实暴力**
- 血腥、写实、逼真
- 例子：包含血液、断肢的画面
- 风险：引发对脱敏和"诡异写实"的担忧
- 接受度：需要年龄分级

### 第二步：分析玩家注意力焦点
评估玩家在游戏中的实际关注点：

**经验丰富的玩家**
- 专注于游戏机制、问题解决、技巧掌握
- "暴力"只是解谜的美学包装
- 关键判断：玩家是在体验暴力还是解决谜题？

**新手玩家**
- 可能更关注视觉冲击
- 需要考虑首次接触时的体验

### 第三步：评估现实区分能力
考虑目标受众的认知水平：
- 正常成年人能够区分幻想与现实
- 儿童需要分级系统保护，尚未为特定概念做好准备
- 评估玩家是否能理解游戏与现实的界限

### 第四步：道德边界检查
作为设计师的责任：
- 监控技术进步带来的新极端机制
- 确保不跨越无形的道德红线
- 考虑长期社会影响和心理影响

## 输出
生成风险评估报告，包含：
1. 暴力类型分类（抽象/卡通/图形）
2. 潜在的道德和心理风险等级
3. 建议的年龄分级
4. 针对不同受众群体的建议
5. 设计调整建议（如需要）
