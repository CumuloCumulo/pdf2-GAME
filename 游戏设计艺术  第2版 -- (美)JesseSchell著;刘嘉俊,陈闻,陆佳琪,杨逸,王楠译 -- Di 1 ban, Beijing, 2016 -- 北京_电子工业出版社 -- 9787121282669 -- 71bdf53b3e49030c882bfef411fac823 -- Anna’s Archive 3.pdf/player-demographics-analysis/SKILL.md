# Player Demographics Analysis

## 何时使用
- 设计针对特定年龄组或性别的游戏时
- 分析玩家体验和参与度时
- 开发多人游戏功能时
- 评估玩家技能需求和难度平衡时

## 执行步骤

### 1. 分析玩家技能类型
列出游戏要求玩家练习的所有技能，并进行分类：

**身体技能**
- 涉及力量、灵巧度、协调性和耐力
- 适用于体育类游戏、舞蹈游戏、控制器操作

**心智技能**
- 涉及记忆、观察和谜题解决
- 适用于策略游戏、解谜游戏

**应用心流状态条件**：确保玩家的技能水平与游戏难度匹配，使玩家保持在"心流"区域

### 2. 确定年龄组特征
参考九年龄组分类系统评估目标受众特点：

- **0-3岁**：对玩具感兴趣但游戏难度过大，适合直接界面（触摸屏）
- **4-6岁**：首个对游戏感兴趣的年龄，常与父母一起玩
- **7-9岁**："启蒙期"，能阅读、思考解决问题，开始自主选择
- **10-13岁**："痴迷期"，生理变化大，沉迷爱好
- **13-18岁**：准备成年，男孩偏好竞争/征服，女孩更关注现实问题/沟通
- **18-24岁**：首个成年组，仍是主要消费群体
- **25-35岁**：家庭建设期，"核心玩家"是重要目标市场
- **35-50岁**：家庭成熟期，寻求全家游戏体验
- **50+岁**：空巢期，偏好社交体验，需要简洁界面

**关键原则**：与特定年龄群体的童年游戏和主题保持一致

### 3. 考虑性别偏好

**男性玩家偏好**
1. 掌控 - 喜欢掌握事物，即使不重要
2. 竞争 - 喜欢与他人竞争以证明自己最强
3. 破坏 - 喜欢破坏事物
4. 空间谜题 - 通常空间推理能力更强
5. 试错 - 偏好通过尝试来学习

**女性玩家偏好**
1. 情感 - 享受探索丰富的人类情感
2. 现实连接 - 偏好与现实世界相关的娱乐
3. 培育 - 享受照顾/培育事物
4. 对话和文字谜题 - 通常语言能力更好
5. 示例学习 - 偏好通过清晰的教程逐步学习

### 4. 应用Bartle玩家类型分类
使用四象限模型设计多人游戏功能：

| 类型 | 主要乐趣 | 焦点 |
|------|----------|------|
| 成就者 | 挑战 | 对世界施加影响 |
| 探索者 | 探索 | 与世界互动 |
| 社交者 | 友谊/合作 | 与其他玩家互动 |
| 杀手 | 竞争/破坏 | 对其他玩家施加影响 |

**设计问题**：
- 每种类型有哪些吸引他们的功能？
- 某种类型是否被忽视？
- 不同类型如何互动？
- 游戏能否平衡所有类型的吸引力？

## 输出结果
- 针对目标人群优化的游戏设计决策
- 匹配年龄和性别偏好的功能列表
- 满足多种玩家类型的功能设计
