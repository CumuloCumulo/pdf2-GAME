# Scott McCloud 投射理论

## 核心概念

Scott McCloud 在《理解漫画》中提出了角色抽象与读者投射的关系。

## 抽象连续体

```
写实照片 -------- 简化漫画 -------- 抽象符号
    |                  |                  |
  观察对象          识别对象          概念理解
    |                  |                  |
  低投射性          中等投射性        高投射性
```

## 理论要点

### 1. 简化增强认同
当角色面孔被简化时，读者更容易将其视为"人"而非"特定的人"，从而更容易投射自己。

### 2. 空白是邀请
空白和模糊的细节邀请读者填充自己的想象。

### 3. 概念 vs 个体
- **具体细节**：定义了一个特定的个体
- **抽象符号**：代表了一个概念或类别

## 游戏应用

### 成功案例

**《传送门》Chell**
- 几乎无对话
- 面部细节极少
- 玩家完全投射自己

**《勇者斗恶龙》系列主角**
- 经典的沉默主角
- 简单的像素设计
- 玩家自由定义性格

**《风之旅人》**
- 完全无对话
- 抽象的斗篷形象
- 纯粹的体验投射

### 失败案例

**过度定义的玩家角色**
- 强制性格
- 固定外貌
- 玩家无法认同

## 视觉对比参考

### 类比：简单素描 + 高清照片

就像在高清照片上叠加简单素描：
- 照片提供丰富的世界细节
- 素描提供可投射的角色符号
- 对比产生独特的视觉冲击

### 游戏中的实现

**《蔚蓝 Celeste》**
- 像素风格主角（简单）
- 精细的背景和光照（复杂）
- 强烈的视觉对比

**《死亡细胞 Dead Cells》**
- 简单的无头角色
- 极其精细的环境和特效
- 玩家聚焦于动作而非角色身份

## 设计决策树

```
开始：设计玩家角色
  |
  |-- 玩家想成为英雄？
  |    |-- 是 --> 理想化类型
  |    |        |-- 增加视觉吸引力
  |    |        |-- 强化能力展示
  |    |
  |    |-- 否 --> 玩家想投射自己？
  |             |-- 是 --> 空白板类型
  |             |        |-- 减少面部细节
  |             |        |-- 保持沉默或中性
  |             |
  |             |-- 否 --> 混合类型
  |                      |-- 理想化能力 + 符号化面孔
  |                      |-- 示例：超级英雄戴面具
```

## 心理学基础

### 镜像神经元
玩家通过控制角色激活镜像神经元，产生"我就是角色"的体验。

### 空白板效应
信息缺失促使大脑自动填充，创造更强的参与感。

### 自我扩张理论
玩家通过角色体验扩展自我边界。
