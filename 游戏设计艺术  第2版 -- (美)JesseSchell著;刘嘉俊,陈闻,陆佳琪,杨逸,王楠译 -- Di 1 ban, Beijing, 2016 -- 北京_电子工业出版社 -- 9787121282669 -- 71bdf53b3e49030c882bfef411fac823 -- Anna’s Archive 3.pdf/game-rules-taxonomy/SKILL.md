# 游戏规则分类法

## 适用场景
- 设计桌面游戏、电子游戏或体育运动规则
- 分析现有游戏的规则结构
- 解决规则冲突或漏洞

## 执行步骤

### 1. 按权威/来源分类
- **官方规则**：由游戏发行商或管理机构制定的正式规则。
- **自定规则**：玩家群体自行约定的地方性规则。
- **不成文规则**：约定俗成但未明文规定的规则。
- **成文规则**：明确记录在案的规则。
- **法律**：具有强制约束力的正式规定。

### 2. 按功能分类
- **基础规则**：游戏的核心基础，决定游戏本质。
- **操作规则**：具体玩法和操作流程。
- **行为规则**：玩家行为准则和礼仪。
- **建议性规则**：策略建议和最佳实践。

### 3. 分析结构
- 识别游戏模式和子模式。
- 确定规则在不同模式下的适用性。

### 4. 应用分析框架
使用Parlett规则分析法等框架进行深入分析。

### 5. 定义执行机制
- 确定违规的判定方式。
- 设定惩罚措施。
- 明确执行主体（裁判、系统、玩家共识）。

## 输出格式
一套按来源和功能分类的综合规则集，包含结构分析和执行机制说明。
