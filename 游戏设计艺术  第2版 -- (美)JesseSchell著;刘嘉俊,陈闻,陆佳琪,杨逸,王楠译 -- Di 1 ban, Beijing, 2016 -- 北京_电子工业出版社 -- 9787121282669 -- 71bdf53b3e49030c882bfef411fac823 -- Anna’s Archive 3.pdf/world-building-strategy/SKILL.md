# 世界构建策略：细节与想象平衡

## 核心原则

世界感觉丰富详细，但无需超出预算或因低质量资产破坏沉浸感。

## 五大策略

### 1. 质量阈值

只有质量足够高时才实现细节。

**规则**：如果实现质量低（如糟糕的配音、低分辨率纹理），留给玩家想象。

**案例**：
- 使用字幕而非低质量语音合成
- 玩家会想象更好的声音

### 2. 功能性细节

提供帮助玩家理解机制或角色的细节。

**案例**：
- 国际象棋棋子设计为中世纪角色（骑士、国王）
- 帮助玩家记住移动规则

### 3. 利用熟悉度

对于玩家熟悉的场景（城市、房屋），提供最少细节。

**原理**：想象力会高效填充其余部分。

### 4. 望远镜效应

1. 早期提供高细节特写建立记忆
2. 后期切换到低细节视图（精灵图）
3. 依靠初始记忆和想象填补差距

### 5. 刺激想象

提供唤起性细节，让玩家轻松投射自己的幻想。

**案例**：「指挥皇家军队」——玩家自行填充宏大感

## 决策矩阵

| 资产质量 | 玩家熟悉度 | 策略 |
|----------|------------|------|
| 高 | 高 | 适度细节，强化关键特征 |
| 高 | 低 | 更多细节，引导理解 |
| 低 | 高 | 最少细节，依赖想象 |
| 低 | 低 | 必要细节，保持抽象 |

## 执行检查清单

- [ ] 评估资产实现质量
- [ ] 判断玩家对主题的熟悉程度
- [ ] 确定哪些细节具有功能性作用
- [ ] 决定是否使用望远镜效应
- [ ] 测试玩家想象是否正确填充空白
